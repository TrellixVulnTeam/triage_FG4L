Once that a deferable event is defined for a state, it cannot be removed.

Scenario: (precond. a state is defined with a deferable event)

- open the "Add event" Dialog in the "Deferable event" section of the properties
panel for the state
- select the event in the right part of the dialog
- press the button with the '<' arrow to remove it, the event correctly goes in
the left part of the dialog
- press ok to confirm the operation and close the dialog

result: the event still appears in the deferable events for the state
